# Recipe: bring-your-own

The empty recipe. For users who know their stack and want Syntaris's methodology without a pre-baked stack opinion.

## When to use

- You already have a stack you trust
- You want methodology (gates, memory, calibration) without stack opinions
- Your stack isn't yet covered by a Syntaris recipe and you don't want to write one

## What this recipe ships

Empty placeholder files. The `/build-rules` interrogation will ask you to fill in:
- Tech stack
- Coding rules
- Deployment platform
- Code patterns
- Banned technologies (optional)

Your answers become the foundation of CONTRACT.md.

## When NOT to use

- You're a casual coder who isn't sure what stack you want - pick `web-app-starter` or `python-cli` instead
- You want to validate your stack against community-tested recipes - pick a populated recipe and override what you need

## Time to first gate

Slightly longer than a populated recipe. Expect 30-60 extra minutes during the CONFIRMED interrogation because the agent has no defaults to fall back on.
